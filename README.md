# Hand Gesture Recognition using OpenCV and MediaPipe

## SkillCraft Technology - Machine Learning Internship

### Task 04

This project performs real-time hand gesture recognition using OpenCV and MediaPipe.

## Features

- Real-time webcam detection
- Multi-hand tracking
- Finger counting
- Gesture recognition
- Landmark visualization
- FPS display
- Interactive UI

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- Jupyter Notebook

## Recognized Gestures

- Fist
- Open Hand
- Thumbs Up
- Peace Sign
- Pointing
- Call Me
